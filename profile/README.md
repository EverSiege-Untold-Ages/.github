## About EverSiege: Untold Ages

EverSiege: Untold Ages is a strategy action game that blends base building, tactical combat, and survival mechanics across evolving battlefields. Players construct defenses, deploy units, and engage in real-time battles against waves of enemies while adapting to changing conditions and increasing difficulty.

The game focuses on planning, positioning, and resource management. Players must build structures, upgrade defenses, command forces, and respond to enemy attacks with efficient strategies. Each stage introduces new threats, requiring adjustments in tactics and base design.

With a mix of action, tower defense mechanics, and progression systems, EverSiege: Untold Ages offers a dynamic gameplay experience built around survival, strategy, and continuous improvement.
